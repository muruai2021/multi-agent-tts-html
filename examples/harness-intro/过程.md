## 💡 经验总结

1. **审核驱动开发**：v1.0.0 → v1.4.0 共 5 个版本，每个版本都来自具体失败案例
2. **CSS 特异性战争**：修复 A bug 时加的兜底规则可能引发新冲突，应让 JS 负责状态、CSS 只负责渲染
3. **HyperFrames 用 GSAP**：不能写 imperative `audio.play()`，必须注册 `window.__timelines`
4. **浮点重叠**：相邻 cue 时长精确衔接会触发 lint 错误，留 50ms gap 是稳妥做法
5. **字幕规范先行**：42px + 三层 text-shadow + 禁容器 = harness-intro 系列的品牌语言
